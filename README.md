## Local Development

Simply open `index.html` in your browser. No build process required!

## Deployment

### Deploy to Vercel

1. Push to GitHub
2. Import repository in Vercel
3. Deploy automatically

### Deploy to Netlify

1. Drag and drop the project folder to Netlify
2. Configure custom domain (optional)
